Atomic Plugins Docs
===================

This repository generates all the Atomic plugins docs. Just run:

npm install
gulp doc

And all the docs will be generated in the dist output directory.

There are independent targets for each language:
- doc-js
- doc-cpp
- doc-android
- doc-ios
